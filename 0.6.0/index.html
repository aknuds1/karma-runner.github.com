Spectacular Test Runner for JavaScript

On the AngularJS team, we rely on testing and we always seek for better tools that would make our life easier. That's why we created Testacular - a test runner that fits all our needs.

Designed for Quick Development

Things should be simple. We believe in testing and so we wanna make it as simple as possible.

The main goal for Testacular is to bring a productive testing environment to developers. An environment, where they don't have to set up many things and rather just write the code and get an instant feedback. Because getting a quick feedback is what makes you productive and creative.

  • Open Source

    Developed for and maintained by the open source community at GitHub.

  • Remote Control

    Control the whole workflow from the command line or your favourite IDE. Just save a file and Testacular will run all the tests.

  • Test on Real Devices

    Test your code on real browsers and real devices such as phones, tablets or headless PhantomJS.

  • Easy Debugging

    Easy debugging directly from your IDE (WebStorm).

  • Mocha, QUnit & Jasmine Adapters

    Testacular is framework agnostic. However it ships with adapters for Mocha, QUnit and Jasmine, as well as AngularJS's scenario runner.

  • Continous Integration

    Simple integration with Jenkins, Travis or Teamcity.

For more motivation on why to use Testacular and why we did it, check out the blog post on Google Testing.
